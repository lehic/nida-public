Kit logo autonome pour usage externe (press, partenaires, réseaux sociaux, print).

## Règles d'usage

### Clear space
Réserver autour du wordmark l'équivalent de la hauteur de la lettre « n ».

### Taille minimale
- Digital : 80px de largeur minimum
- Print : 20mm de largeur minimum

### Couleurs
| Variante | Fond conseillé |
|---|---|
| Forest `#163300` | Fonds clairs (#F8F6F0, #FFFFFF) |
| Gold `#D4A843` | Forest `#163300`, noir |
| White `#FFFFFF` | Photos sombres, forest |
| Mono black | Documents print, fax, mono |
| Mono white | Fonds foncés en mono |

### Ne jamais
- Déformer, incliner, ajouter d'ombre/contour
- Modifier les proportions entre lettres / kaaba / hamza
- Utiliser sur fonds aux contrastes insuffisants (<4.5:1)
- Recréer le wordmark en tapant « nida » dans une autre police
- Utiliser la majuscule : toujours lowercase
